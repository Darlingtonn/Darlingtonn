Soy estudiante de desarrollo con enfoque en **Python, Django, SQL, bases de datos y aplicaciones web**. En este portafolio presento proyectos académicos y prácticos orientados a construir soluciones tecnológicas funcionales, documentadas y fáciles de ejecutar.

- Desarrollo web con Python y Django.
- Construcción de aplicaciones CRUD.
- Autenticación de usuarios con Django Auth.
- Modelamiento de bases de datos relacionales.
- Consultas SQL, transacciones y control de integridad.
- Uso de Git y GitHub para control de versiones.
- Maquetación web con HTML, CSS y JavaScript.

## Proyectos destacados

### E-commerce Django para Portafolio

Aplicación Django final que integra catálogo de productos, autenticación, vista protegida y gestión CRUD de productos.

**Tecnologías:** Python, Django, SQLite, HTML, CSS, Django Auth.  
**Repositorio:** https://github.com/Darlingtonn/ecommerce-django-portafolio

### Base de Datos Relacional para E-commerce

Modelo SQL para administrar usuarios, categorías, productos, stock, pedidos y detalle de pedidos. Incluye scripts de creación, datos de prueba, consultas y transacciones.

**Tecnologías:** SQL, PostgreSQL, modelamiento relacional.  
**Repositorio:** https://github.com/Darlingtonn/ecommerce-sql-modulo5

### E-commerce con Autenticación en Django

Proyecto Django enfocado en registro de usuarios, inicio de sesión, cierre de sesión y acceso a vistas protegidas.

**Tecnologías:** Python, Django, Django Auth, HTML, CSS.  
**Repositorio:** https://github.com/Darlingtonn/ecommerce-django-auth-modulo6

### CRUD de Productos con Django ORM

Proyecto Django orientado a crear, listar, editar y eliminar productos usando modelos, formularios, validaciones, templates y ORM.

**Tecnologías:** Python, Django ORM, SQLite, HTML, CSS.  
**Repositorio:** https://github.com/Darlingtonn/ecommerce-django-crud-modulo7
